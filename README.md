# 🧠 ML-Based Skin Disease Detection System

**📅 Apr 2026**

---

## 🚀 Project Overview

This project is a Machine Learning-based system that detects and classifies different types of skin diseases from input images.
It uses a Convolutional Neural Network (CNN) to analyze skin images and predict the disease category with good accuracy.

---

## 🎯 Key Features

* Upload skin images for prediction
* Detect and classify multiple skin diseases
* Clean and simple user interface
* Fast and reliable predictions using trained CNN model

---

## 🧠 Model Details

* Built using Convolutional Neural Network (CNN) architecture
* Preprocessed and normalized image data for consistent input
* Used image augmentation to improve model performance
* Trained on labeled dataset of skin disease images
* Evaluated using standard metrics like accuracy and loss

---

## 📊 Workflow

1. Collect and preprocess image dataset
2. Normalize and resize images
3. Train CNN model on dataset
4. Evaluate model performance
5. Save trained model (`.h5`)
6. Use model for real-time prediction
